# Customer Churn Prediction

## Objective
Predict whether a telecom customer will churn (1) or stay (0).

## Dataset
Telco Customer Churn Dataset (7043 customers)

## Steps Performed
- Data Cleaning
- Handling Missing Values
- Encoding Categorical Variables
- SMOTE for Class Imbalance
- Logistic Regression Model
- Model Evaluation

## Model Performance
- Accuracy: 79%
- Recall (Churn): 83%
- F1 Score: 0.80
- ROC-AUC Score: 0.87

## Business Impact
Helps telecom companies identify high-risk customers and take retention actions.
